# SE-Course-Capstone
Capstone project


A webpage to search for datasheets based on either search filters or by a specific part number.
- Similar to https://toolsearch.exsysautomation.com/default/index/en, but with far less data in the database.
- Custom database would be necessary to create.
- Simple single page site with a login to save your selected datasheets.


Use schema data from newDrawing.js to create new drawings and save them to the db.

You cannot enter new or view drawings without signing in first.
